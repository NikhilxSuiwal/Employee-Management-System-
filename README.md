# React Employee Management App

## About Project
This is a React-based Employee Management System where admin and employees can manage tasks.

##  Tech Used
- React.js
- JavaScript
- Tailwind CSS
- LocalStorage

## Features
- Login system (Admin & Employee)
- Task viewing dashboard
- Task categories and details display

## How to Run
1. npm install
2. npm run dev
